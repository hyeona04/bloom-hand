# Bloom Hand

손바닥을 인식하면 화면에 꽃 애니메이션이 나타나는 인터랙티브 웹 프로젝트입니다.

## Tech Stack
- React
- Vite
- TypeScript
- Tailwind CSS
- Framer Motion
- MediaPipe Tasks Vision

## Features
- 손바닥 인식
- 실시간 애니메이션
- 카메라 기반 인터랙션

## Run
```bash
npm install
npm run dev
