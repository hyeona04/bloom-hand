# 🌸 Bloom Hand

## 📌 한 줄 소개
손바닥을 인식하여 실시간으로 인터랙티브 애니메이션을 출력하는 웹 프로젝트

---

## 🔗 배포 URL
👉 https://frabjous-griffin-889d52.netlify.app/

---

## 🛠 Tech Stack

- React (Vite)
- TypeScript
- Tailwind CSS
- Framer Motion
- MediaPipe Tasks Vision

---

## ✨ 핵심 기능

- 손바닥 인식 기반 인터랙션
- 실시간 애니메이션 출력
- 손 위치 기반 효과 발생
- 손을 오래 펼치면 더 강한 효과 출력
- 카메라 입력을 활용한 사용자 참여형 UX

---

## 💡 구현하면서 어려웠던 점

### 1. 손 좌표와 화면 좌표 매칭
MediaPipe에서 제공하는 정규화 좌표(0~1 범위)를  
실제 화면(Canvas) 좌표로 변환하는 과정에서  
정확한 위치에 애니메이션을 출력하기 어려움

### 2. 좌우 반전 문제
카메라 입력과 사용자 인식 방향이 달라  
손의 움직임과 화면 반응이 어긋나는 문제 발생

---

## 🔧 해결 방법

- 좌표를 Canvas 크기에 맞게 변환하여 위치 정확도 개선
- `x → (1 - x)` 방식으로 좌우 반전 처리
- 손 인식 threshold를 조정하여 안정적인 트리거 구현
- 프레임 단위 업데이트로 자연스러운 애니메이션 흐름 유지

---

## 📈 추가 개선 아이디어

- 모바일 환경 최적화
- 다양한 인터랙션 모드 추가 (빛, 파티클 등)
- 배경 및 테마 변경 기능
- 손 제스처 확장 (주먹, 핀치, 스와이프 등)

---

###손바닥을 인식하면 실시간으로 애니메이션이 반응하는 인터랙티브 웹 프로젝트를 구현했습니다.  
MediaPipe를 활용해 손의 랜드마크 좌표를 추출하고, 이를 Canvas 좌표로 변환하여 화면과 정확히 매칭했습니다.  
특히 카메라 좌우 반전 문제로 인해 사용자 움직임과 화면 반응이 어긋나는 문제가 있었는데,  
x 좌표를 반전하는 방식으로 해결해 직관적인 인터랙션을 구현했습니다.  
이를 통해 웹 기술과 컴퓨터 비전 기술을 결합한 실시간 사용자 경험을 구현할 수 있었습니다.
